# GitHub

Der »[Regenpfeifer](https://regenpfeifer.net/kontakt)« ist aktiv hauptsächlich auf seinen [anderen Webseiten](https://regenpfeifer.github.io) anzutreffen. Auf GitHub verwaltet er die technischen Hilfsmittel für seine inhaltliche Arbeit: Skripte für Linux (Ubuntu) und Anpassungen für ZSH sowie Vorlagen für LibreOffice und LaTeX. Außerdem dient ihm GitHub zur Versionsverwaltung und Sicherung seiner mit LaTeX erstellten Texte.

Die Aktivitäten des »Regenpfeifers« sind [hier](https://github.com/regenpfeifer) zu finden. Bislang sind alle seine eigenen Repositories privat und nicht für die Öffentlichkeit bestimmt.Er beteiligt sich aber gelegentlich an Projekten anderer Entwickler.
